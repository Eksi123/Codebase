Chapter2

超链接-页面跳转

(1) 同级目录下链接：< a href="文件名" > “点击内容” < /a >

(2) 上级目录html文件链接下级目录文件：< a href="下级目录名/文件名" > “点击内容” < /a >
或者：< a href="下1级目录名/下2级目录名/文件名" > “点击内容” < /a >

(3) 下级目录html文件链接上级目录文件：< a href="../文件名" > “点击内容” < /a >
或者：< a href=".. /上1级目录文件名/文件名" > “点击内容” < /a >
